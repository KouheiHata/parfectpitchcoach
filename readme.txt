【wordpress：パーマリンク設定】
数字ベース

【テキストの太字と色】
参考元がそのような表現が無かったため
太字だけ反映したものもありますが、
ほとんど装飾無しのテキストで記述してあります。
後から指摘されたら反映しようと思っておりまして、
そのままにしております。

【レスポンシブ対応（スマホ対応）】
Bootstrap4というcssとjavascriptを使ったものを導入しています。
header.phpとfooter.phpに導入するために記述があります。

【注文や問い合わせフォーム】
プラグイン[ContactForm 7]というものを導入して
wordpressのメニュー「お問い合わせ」に表示部を登録しています。
ContactForm7に登録したソースコードが[ContactForm7]フォルダに作られています。

【リンク】
各PHPは固定ページに登録して使うことを前提にしております。
アドレスはそこで任意に決められるので、
PHP内にあるリンクは正式なアドレスが分からないので設定しておりません。。

【移植の約束事】
参考元サイトに習って以下を考慮して作られています。
・見出しは３種類。ボタンは２種類。
・写真の右にはテキストを置かないで下に。
・コンテンツを横に並べない。
